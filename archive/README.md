# Archive Directory

This directory contains legacy files, old documentation, and unused code that are preserved for historical reference but are no longer part of the active codebase.

## Contents

### `/archive/docs/`
Development planning documents, deployment guides, and historical documentation that were previously in the project root:
- Development plans and sprint documents
- Deployment guides for specific servers
- Feature planning and tracking documents
- Migration logs and notes
- Testing documentation (superseded by docs/TESTING_GUIDE.md)

### `/archive/scripts/`
Legacy utility scripts and diagnostic tools that are no longer actively used:
- One-off diagnostic scripts
- Legacy setup and organization scripts
- Server diagnostic helpers

### `/archive/tests/`
Old diagnostic and verification test scripts:
- Legacy diagnostic tests
- One-off verification scripts
- Quick test utilities (superseded by organized test suite)

## Note

Files in this archive are **not deleted** - they are preserved for reference. However, they are not maintained and may not work with the current codebase structure.

For current documentation, refer to the `docs/` directory.
For active scripts, refer to the `scripts/` directory.
For current tests, refer to the `tests/` directory.
